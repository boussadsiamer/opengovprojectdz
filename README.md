# Algeria OpenGov

A transparency platform for visualizing government budgets, regional spending, and national projects in Algeria.

## Features

- Budget visualization and sector breakdown
- Regional spending maps and data
- National projects tracker
- Citizen feedback system
- Interactive charts and visualizations

## Technologies

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Chart.js
- Leaflet.js

## About

This project was created for a school project to demonstrate open government concepts and data visualization.
